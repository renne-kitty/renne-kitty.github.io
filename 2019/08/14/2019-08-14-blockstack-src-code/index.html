blockstack源码结构

用户交互模型

blockstack naming system简称BNS，是blockstack用于处理用户需求的名称系统。

用户在访问BNS时，一般通过调用BNS API，实现对BNS的调用。而BNS API会使用私有的方法，对虚拟链进行调用，然后访问gaia存储，从而实现各种各样的业务逻辑。

blockstack-core

blockstack-core是blockstack核心源码，它主要包括了三部分：

  1. 提供用户接口的API
  2. atlas网络(类p2p网络)的实现
  3. 虚拟链hook的实现(实现对虚拟链各种信息的读取、验证)

virtualchain

virtualchain描述了虚拟链部分的逻辑，其主要包括两部分：

  1. 底层区块链的接口，这里只运用bitcoin作为接口
  2. 虚拟链区块结构
