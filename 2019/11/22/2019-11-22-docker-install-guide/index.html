linux下docker安装及运行

摸索docker这东西折腾了我一段时间，果然掌握的基本工具还真的是太少了，我装的是ubuntu最精简版的docker。一开始由于各种网络出现问题，也不知道为什么，然后就没怎么管它了，结果到现在碰到硬需求，就硬着头皮上吧。

首先安装docker，很简单不解释

安装好之后需要在root用户下启动并运行

之后要做的就是拉去镜像，使用search搜索镜像，然后用pull拉拉取下来

这时我们已经有了ubuntu精简版的镜像，我们需要将它放后台运行，不然它网络不会启动，我现在还不太知道为什么

运行了之后就产生了一个新的容器(container)，接着系统在容器里运行，我们通过docker exec就可以随便接入。

另外，这时由于只安装了很少软件，所以最好先换个源，安装vim，再执行之后的操作。

接着在docker内部运行apt-get update，再下载个vim就完美了。
