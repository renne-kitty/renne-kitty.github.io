安卓8.1安装记录(vmware虚拟机)

安装镜像

我这里使用的是安卓8.1，自己到官网上下载x86版的镜像，我选了32位的版本。

镜像就像平常一样安装，分区格式化就跟linux安装一样，这种事应该安装过linux都能解决。安装好之后，会发现只有命令行系统，没有我们熟悉的安卓界面。

这时需要更换命令行。在选中第一项，也就是没有任何特殊选项的那一项按e，编辑命令行。

将"quiet"删掉，并把root改成nomodeset_root。其实我也不知道为什么要这样干，应该是启动时执行的模式吧。

之后随便填写东西就能正常进入系统了。

网络连接

由于我主要用的是wifi网络，而且我存在要让这台安卓的ip地址与本机wifi的ip地址设置在同一网段的需求。因此我使用了桥接模式，由于使用了多网卡，因此在使用的时候，需要将桥接的网卡更换为本机的无线网卡。

在安卓里面使用虚拟的无线连接，我这里名字叫"Virtwifi"。

p.s. 刚刚试过一下，可能该镜像仅支持2.4GHz的网络，因此连接5GHz的网络可能会出问题。

设置好以后，发现wifi上面有个叉叉，虽然我们上baidu.com是没问题的，但还是写着已连接但不能访问网络的字样。这是因为原生安卓在连接wifi的时候都会向谷歌某网站发送一个https的请求，该网站只会返回一个code=204的空白网页，在国内由于你懂的原因无法访问。因此我们用adb将这个地址改为国内的即可。

大概就是改一个环境变量的意思。网上还有dalao直接改源码的，但安卓8.1已经提供了环境变量供更改设置就不用这样烦了。
